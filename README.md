# UniGuard
(Uniform + Guard)

현재 레포지토리는 프론트 코드만 있음

### 프로젝트 소개
AI 이미지 인식 기술을 활용한 교복·체육복 자동 점검 웹 시스템
 실시간 복장 인식 결과를 웹 대시보드로 확인
 위반 감지 시 알림 전송 및 인식 기록 관리
 학교 생활 안전 관리의 효율성 향상 목적
* 플랫폼 : 웹
* 제작 인원 : 1인(개인 프로젝트)
* 제작 기간 : 10일 (2025.12.17 ~ 2025.12.26)
* 사용 기술 : HTML,CSS,JS,Python,FastAPI,YOLOv11,MySql,AWS(RDS)

### 기능 
* 실시간 인식
* 인식 기록
* 알림 시스템

### 아쉬운 점
볼륨 디스크 용량 부족 때문에 Yolo로 만든 AI모델을 EC2에 못 올려서 배포를 못 한 점이 아쉽다. 

### 느낀 점
처음 강의 들을 때 AWS가 뭔지도 잘 몰랐습니다. 강의를 듣고나니 AWS 기초가 잡힌 것 같고 IP간의 이동, 클라이언트와 서버간의 호출, 응답을 깊게 이해할 수 있어서 뜻깊은 시간이였습니다. 
앞으로 프로젝트 개발과 발표를 할 때 시스템 아키텍처를 꼭 만들어야 겠다 생각이 들었고 아키텍처를 잘 만들어갸 겠다, 기획을 잘 해야겠다 라는 생각이 들었습니다. 
배포는 못 했지만 프로젝트 완성은 해서 뿌듯하고 보람찼습니다.
